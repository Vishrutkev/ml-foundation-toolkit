# Machine Learning Foundations Toolkit

A portfolio-ready repository showcasing classical machine learning algorithms, optimization methods, and vectorized scientific computing in Python.

## Highlights
- k-nearest neighbors for classification and regression
- vectorized Euclidean distance computation with NumPy
- gradient descent with JAX autograd
- linear regression and Ridge regression
- mini-batch SGD vs closed-form optimization on MNIST
- PCA and LDA for dimensionality reduction
- KNN comparison across transformed feature spaces

## Suggested resume entry

**Machine Learning Foundations Toolkit**  
*Python, NumPy, JAX, scikit-learn, Matplotlib*  
- Implemented classical machine learning algorithms from scratch, including linear regression with gradient descent, Ridge regression, k-NN, and vectorized Euclidean distance computation.  
- Compared optimization methods such as closed-form solutions, batch gradient descent, and mini-batch SGD on supervised learning tasks including MNIST digit classification.  
- Applied PCA and LDA on real-world datasets to analyze dimensionality reduction, class separability, and downstream KNN classification performance.  
- Improved computational efficiency by replacing loop-based numerical operations with NumPy vectorization and broadcasting techniques.
